Python Web Tutorial


David Yang / 2019 Jan - 2020 May

Python Web Development Full Stack Tutorial Shell Script Search

This tutorial is a culmination of Mr. Liao Xuefeng's 2016 Python 3 tutorial, and enriched with the latest frontend technologies. Instead of relying on established and convenient Python web development frameworks like Flask or Django, this tutorial takes a ground-up approach, guiding readers through building a web application from scratch.


The tutorial covers a wide range of topics, starting with setting up the database, constructing the web framework, implementing the ORM (Object-Relational Mapping), establishing the MVC (Model-View-Controller) pattern, creating APIs, and finally crafting frontend pages using CSS/HTML, JavaScript, and DOM manipulation. Moreover, it includes detailed code examples and thorough explanations to ensure readers have a comprehensive understanding of each step in the website development process.






*Note: This tutorial is original in Chinese and finished independently.
