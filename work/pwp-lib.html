PWP Asset Library - Asset Management System


David Yang / 2020 April

Digital Asset Management React Redux Python Shell Script

This project aims to address the challenge of slow and incomplete asset searches on the company server by implementing an efficient asset management system. Utilizing Python and batch scripting, I effectively tracked and processed asset updates, while employing React and Redux to create a responsive user interface. This system significantly enhanced the efficiency of the design process, receiving positive feedback and appreciation from my colleagues.
